# Vite Starter

A minimal Vite + TypeScript starter for the Pubky tutorial.

It includes `@synonymdev/pubky@0.9.3`, but no Pubky app code yet.

## Quick Start

```bash
npx tiged pubky/pubky-app-templates/vite-starter my-pubky-app
cd my-pubky-app
npm install
npm run dev
```
